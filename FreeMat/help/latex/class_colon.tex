\section{COLON Overloaded Colon Operator}

\subsection{Usage}

This is a method that is invoked in one of two forms, either
the two argument version
\begin{verbatim}
  c = colon(a,b)
\end{verbatim}
which is also called using the notation
\begin{verbatim}
  c = a:b
\end{verbatim}
and the three argument version
\begin{verbatim}
  d = colon(a,b,c)
\end{verbatim}
which is also called using the notation
\begin{verbatim}
  d = a:b:c
\end{verbatim}
