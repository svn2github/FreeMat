\section{CTRANSPOSE Overloaded Conjugate Transpose Operator}

\subsection{Usage}

This is a method that is invoked when a variable has the
conjugate transpose operator method applied, and is invoked
when you call
\begin{verbatim}
   c = ctranspose(a)
\end{verbatim}
or
\begin{verbatim}
/  c = a'
\end{verbatim}
