\section{GE Overloaded Greater-Than-Equals Comparison Operator}

\subsection{Usage}

This is a method that is invoked to combine two variables using a
greater than or equals comparison operator, and is invoked when you call
\begin{verbatim}
   c = ge(a,b)
\end{verbatim}
or for 
\begin{verbatim}
   c = a >= b
\end{verbatim}
