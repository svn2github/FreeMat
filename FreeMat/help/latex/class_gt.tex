\section{GT Overloaded Greater Than Comparison Operator}

\subsection{Usage}

This is a method that is invoked to combine two variables using a
greater than comparison operator, and is invoked when you call
\begin{verbatim}
   c = gt(a,b)
\end{verbatim}
or for 
\begin{verbatim}
   c = a > b
\end{verbatim}
