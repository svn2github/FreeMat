\section{MLDIVIDE Overloaded Matrix Left Divide Operator}

\subsection{Usage}

This is a method that is invoked when two variables are divided
using the matrix (left) divide operator, and is invoked when
you call
\begin{verbatim}
   c = mldivide(a,b)
\end{verbatim}
or for
\begin{verbatim}
   c = a \ b
\end{verbatim}
