\section{MRDIVIDE Overloaded Matrix Right Divide Operator}

\subsection{Usage}

This is a method that is invoked when two variables are divided
using the matrix divide operator, and is invoked when you call
\begin{verbatim}
   c = mrdivide(a,b)
\end{verbatim}
or for
\begin{verbatim}
   c = a / b
\end{verbatim}
