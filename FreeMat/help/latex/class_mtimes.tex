\section{MTIMES Overloaded Matrix Multiplication Operator}

\subsection{Usage}

This is a method that is invoked when two variables are multiplied
using the matrix operator and is invoked when you call
\begin{verbatim}
   c = mtimes(a,b)
\end{verbatim}
or for
\begin{verbatim}
   c = a * b
\end{verbatim}
