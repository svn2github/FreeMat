\section{POWER Overloaded Power Operator}

\subsection{Usage}

This is a method that is invoked when one variable is raised
to another variable using the dot-power operator, and is
invoked when you call
\begin{verbatim}
   c = power(a,b)
\end{verbatim}
or
\begin{verbatim}
   c = a.^b
\end{verbatim}
