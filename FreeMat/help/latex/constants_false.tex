\section{FALSE Logical False}

\subsection{Usage}

Returns a logical 0.  The syntax for its use is
\begin{verbatim}
   y = false
\end{verbatim}
You can also create an array of logical ones using
the syntax
\begin{verbatim}
   y = false(d1,d2,...,dn)
\end{verbatim}
or the syntax
\begin{verbatim}
   y = false([d1,d2,...,dn])
\end{verbatim}
