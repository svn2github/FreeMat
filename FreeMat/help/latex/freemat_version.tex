\section{VERSION The Current Version Number}

\subsection{Usage}

The \verb|version| function returns the current version number for
FreeMat (as a string).  The general syntax for its use is
\begin{verbatim}
    v = version
\end{verbatim}
\subsection{Example}

The current version of FreeMat is
\begin{verbatim}
--> version

ans = 
4.1
\end{verbatim}
