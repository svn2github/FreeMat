\section{VERSTRING The Current Version String}

\subsection{Usage}

The \verb|verstring| function returns the current version string for
FreeMat.  The general syntax for its use is
\begin{verbatim}
    version = verstring
\end{verbatim}
\subsection{Example}

The current version of FreeMat is
\begin{verbatim}
--> verstring

ans = 
FreeMat v4.0
\end{verbatim}
