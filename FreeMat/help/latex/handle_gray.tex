\section{GRAY Gray Colormap}

\subsection{Usage}

Returns a gray colormap.  The syntax for its use is
\begin{verbatim}
   y = gray
\end{verbatim}
\subsection{Example}

Here is an example of an image displayed with the \verb|gray|
colormap
\begin{verbatim}
--> x = linspace(-1,1,512)'*ones(1,512);
--> y = x';
--> Z = exp(-(x.^2+y.^2)/0.3);
--> image(Z);
--> colormap(gray);
\end{verbatim}
