\section{ISCELL Test For Cell Array}

\subsection{Usage}

The syntax for \verb|iscell| is 
\begin{verbatim}
   x = iscell(y)
\end{verbatim}
and it returns a logical 1 if the argument is a cell array
and a logical 0 otherwise.
\subsection{Example}

Here are some examples of \verb|iscell|
\begin{verbatim}
--> iscell('foo')

ans = 
 0 

--> iscell(2)

ans = 
 0 

--> iscell({1,2,3})

ans = 
 1 
\end{verbatim}
