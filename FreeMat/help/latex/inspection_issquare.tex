\section{ISSQUARE Test For a Square matrix}

\subsection{Usage}

This function tests to see if the argument is a square matrix.  The 
syntax for \verb|issquare| is
\begin{verbatim}
   x = issquare(y)
\end{verbatim}
and it returns a logical 1 if the argument is size \verb|N x N|
logical 0 otherwise.
