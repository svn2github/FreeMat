\section{DELETE Delete a File}

\subsection{Usage}

Deletes a file.  The general syntax for its use is
\begin{verbatim}
  delete('filename')
\end{verbatim}
or alternately
\begin{verbatim}
  delete filename
\end{verbatim}
which removes the file described by \verb|filename| which must
be relative to the current path.
