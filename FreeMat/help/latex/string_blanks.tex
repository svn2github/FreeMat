\section{BLANKS Create a blank string}

\subsection{Usage}

\begin{verbatim}
    str = blanks(n)
\end{verbatim}
Create a string \verb|str| containing \verb|n| blank charaters.
\subsection{Example}

A simple example:
\begin{verbatim}
--> sprintf(['x0123456789y\n','x',blanks(10),'y\n'])

ans = 
x0123456789y
x          y
\end{verbatim}
