\section{vtkPNGReader}

\subsection{Usage}

 vtkPNGReader is a source object that reads PNG files.
 It should be able to read most any PNG file


To create an instance of class vtkPNGReader, simply
invoke its constructor as follows
\begin{verbatim}
  obj = vtkPNGReader
\end{verbatim}
