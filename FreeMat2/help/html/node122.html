SEED Seed the Random Number Generator

Usage

Seeds the random number generator using the given integer seed. Changing the seed allows you to choose which pseudo-random sequence is generated. The seed takes a single uint32 value: 

  seed(s)
where s is the seed value.

Example

Here's an example of how the seed value can be used to reproduce a specific random number sequence. 

--> seed(32);
--> rand(1,5)
