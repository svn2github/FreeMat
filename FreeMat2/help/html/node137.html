SAVE Save Variables To A File

Usage

Saves a set of variables to a file in a machine independent format. There are two formats for the function call. The first is the explicit form, in which a list of variables are provided to write to the file: 

  save filename a1 a2 ...
In the second form, 
  save filename
all variables in the current context are written to the file. The format of the file is a simple binary encoding (raw) of the data with enough information to restore the variables with the load command. The endianness of the machine is encoded in the file, and the resulting file should be portable between machines of similar types (in particular, machines that support IEEE floating point).

Example

Here is a simple example of save/load. First, we save some variables to a file. 

--> D = {1,5,'hello'};
--> s = 'test string';
--> x = randn(512,1);
--> z = zeros(512);
--> who
  Variable Name      Type   Flags   Size
              D      cell           [1 3]
              s    string           [1 11]
              x    double           [512 1]
              z     float           [512 512]
--> save loadsave.dat
Next, we clear all of the variables, and then load them back from the file. 
--> clear all
--> who
  Variable Name      Type   Flags   Size
--> load loadsave.dat
--> who
  Variable Name      Type   Flags   Size
              D      cell           [1 3]
              s    string           [1 11]
              x    double           [512 1]
              z     float           [512 512]
            ans    double           []
