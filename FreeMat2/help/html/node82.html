EXIST Text Existence of a Variable

Usage

Tests for the existence of a variable. The general syntax for its use is 

  y = exist('varname')
The return is 1 if a variable with the name varname exists in the current workspace and is not empty. This function is primarily useful when keywords are used in function arguments.

Example

Some examples of the exist function. Note that generally exist is used in functions to test for keywords. For example, 

  function y = testfunc(a, b, c)
  if (~exist('c'))
    % c was not defined, so establish a default
    c = 13;
  end
  y = a + b + c;
