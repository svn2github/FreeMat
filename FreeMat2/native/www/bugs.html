Bugs

If you find a bug in FreeMat, please help us fix it by submitting a bug report. We use the bug reporting mechanism from SourceForge. Choose from the following links based on what you need:

  • Request Help - use this link if you need help, or are unsure if some behavior is correct or not. This list is the default one to use.
  • Report a Bug - use this link if you have found a bug. A bug report is most useful if you can provide a small example that exercises the problem.
  • Request a Feature - use this link to request a feature that FreeMat does not currently have.
