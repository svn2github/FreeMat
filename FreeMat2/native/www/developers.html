Developers

Do you enjoy using FreeMat? Would you like to make modifications to the source? Or help develop FreeMat to advance its capabilities? Perhaps you would just like to hack! Developers generally work on the CVS version of FreeMat (as opposed to the released versions). You can access the CVS repository via one of the following mechanisms:

  • Anonymous CVS Access - You can access CVS using the anonymous CVS access feature of SourceForge.net. Follow this link for details.
  • Web based CVS Access - here you can simply browse the source. Look at the FreeMat2 module.
