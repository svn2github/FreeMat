Downloads

The FreeMat releases are hosted on SourceForge using the file release system. Please consider making a donation to the project - it would help offset the costs of developing and maintaining FreeMat for multiple platforms (particularly the non-Linux ones). FreeMat is available in several forms - as a precompiled binary for Windows, a precompiled disk image for Mac OS X, a binary install for Linux and as source code package for other operating systems. I simply cannot provide prebuilt binaries for all of the packaging variants, but I will post links here to binaries as they become available. New releases are made as development warrants. I would suggest you sign up to the mailing list, which is very low bandwidth, and primarily contains release notices.

Source for version 2.0

For most Unix-like environments, FreeMat 2.0 is now available in source code form as FreeMat-2.0.tar.gz You will need to select a SourceForge download mirror that is close to you. See the installation chapter of the manual for build instructions.

Binary Linux Tarball for version 2.0

For those of you who do not have root access to your Linux machine, you can download a tarball of FreeMat FreeMat-2.0-Binary.tar.gz. To install it, just extract the tarball, add FreeMat-2.0/Contents/bin it to your path, and off you go. This version should be fairly distribution-agnostic, but I have only tested it on RedHat variants.

Windows Installer for 2.0

A Windows installer is available here: FreeMat-2.0_Setp.exe You will need to select a SourceForge download mirror that is close to you. I have only tested this installer with Windows 2000/XP. MS Windows 98/ME/NT may or may notwork (I have at least one report of FreeMat working with Windows ME).

Mac OS X Bundle for 2.0

A Mac OS X bundle is now available for FreeMat 2.0. FreeMat is a native Aqua application with an Aqua Look and Feel. You can download FreeMat here FreeMat 2.0 This is not a universal application, although I might think about making the next release universal (if I can get it to work properly). This version requires Mac OS X 10.4 or later to work. If you run 10.3 and need a build of FreeMat, please file a Request For Enhancement here. I have no idea what version of OS X people run FreeMat on, and am guessing 10.4. If not, please let me know.
