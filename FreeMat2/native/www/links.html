Interesting Links

  • Octave - a GPL-ed numerical computing language/environment. Very nice!
  • Scilab - developed by INRIA and ENPC, an advanced numerical computation package. Scilab also has a parallel interface (via PVM).
  • Mathworks - makers of MATLAB, Simulink, etc (all registered trademarks).
  • IDL - home page for IDL, from Research Systems Inc.
  • NETLIB - Repository for a host of great numerical software. FreeMat's core numerical routines are all from NETLIB.
  • FLTK Fast Light ToolKit - the cross-platform toolkit used by FreeMat (pre 2.0RC1) for graphical services.
  • ARPACK Arnoldi Package - Contains routines used by FreeMat to solve sparse eigenvalue problems.
  • UMFPACK Unsymmetric MultiFrontal Method Package - Contains routines used by FreeMat to solve sparse linear problems.
